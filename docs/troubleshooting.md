# Troubleshooting Guide

### Problem: Python not recognized
Solution:
Install Python from https://python.org



### Problem: JSON file missing
Solution:
Application automatically creates the file.



### Problem: Wrong menu option entered
Solution:
Use only numbers. Application validates input safely.



### Problem: Folder not found (File Organizer)
Solution:
Enter a valid folder path only.



### Problem: Backup not visible
Solution:
Check path: `data/backups/`
Backup folders are created automatically.
