# Modules Overview

## calculator.py
Contains basic arithmetic functions.
- add()
- subtract()
- multiply()
- divide()



## notes_manager.py
Handles:
- Create notes
- Save notes to JSON
- Load notes from JSON
- Search, edit, delete
- Export TXT/CSV


## timer.py
Provides:
- Countdown timer
- Stopwatch


## file_organizer.py
Organizes files into subfolders based on extension:
- Images
- Documents
- Videos
- Etc.



## unit_converter.py
Length, weight, temperature conversions


## backup_manager.py
Backup & Restore:
- Create backup folder with timestamp
- Copy JSON files
- Restore backup safely



## utils.py
Reusable helper functions:
- clear screen
- get input safely
- format UI titles
- confirm actions
