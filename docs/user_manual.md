# User Manual

This guide explains how to use each tool inside the Personal Productivity Suite.



## Main Menu Options
1. Calculator Tool
2. Notes Manager
3. Timer & Stopwatch
4. File Organizer
5. Unit Converter
6. Backup & Restore
7. Exit


## 1. Calculator Tool
- Choose an operation
- Enter numbers
- View the result
- Supports basic arithmetic operations


## 2. Notes Manager
- Create notes
- View notes
- Search notes
- Edit notes
- Delete notes
- Export to TXT / CSV
- All notes stored in `data/notes.json`



## 3. Timer & Stopwatch
- Countdown timer (enter seconds)
- Stopwatch (measure elapsed time)


## 4. File Organizer
- Organizes files into category folders
- Images, Documents, Videos, etc.
- Uses file extensions to categorize


## 5. Unit Converter
Length:
- meter ↔ kilometer
- meter ↔ centimeter

Weight:
- kg ↔ grams

Temperature:
- Celsius ↔ Fahrenheit



## 6. Backup & Restore
- Create backup of data files
- Automatically stores backups in `data/backups/`
- Restore saved data when needed



## Exit Application
Choose option 7 to exit.
